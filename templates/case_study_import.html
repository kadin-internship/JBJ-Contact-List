1. Upload files

Accepts PDF and Word (.docx) files. Native Google Docs need to be exported first (in Drive: open the doc → File → Download → Microsoft Word or PDF) -- this importer can't read them directly. Claude will read each file and draft the title, client, sector, and challenges/solution/results -- nothing is saved until you review and confirm below.
